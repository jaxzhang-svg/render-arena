## 🔄 文档更新日志

### v1.1 (2025-01-09)

**主要变更**:

- ✨ 新增智能输入框（灵感选择 + 效果库）
- ✨ 新增 LLM 自动命名功能
- ✨ 新增默认模型配置
- 🔧 优化录制模式（所有 UI 移至顶部导航栏）
- 🔧 优化输入控制台（4 种状态管理）
- 📝 拆分为 4 个独立文档，便于维护和查阅

**详细变更**:

- 移除搜索功能
- 移除卡片分享按钮和悬停预览
- 删除空状态设计（官方精选案例）
- 移除技术栈提示（非技术向用户）
- 新增"PRD 与设计的关系"说明章节

### v1.0 (2025-01-08)

**初始版本**:

- 产品定义与核心功能
- 技术架构设计
- 用户旅程规划
- 页面功能示意图
