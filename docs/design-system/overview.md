# Novita Web UI Kit - Design System Documentation

## Overview

This documentation captures the design specifications from the Figma UI Kit for the Novita web application. The design system uses the TT Interphases Pro font family and a mint-green color palette.

## Design Principles

- **Primary Font**: TT Interphases Pro (with TT Interfaces Mono as secondary)
- **Primary Color**: Mint Green (#23D57C)
- **Design Philosophy**: Clean, modern interface with clear visual hierarchy
- **Icon Library**: Phosphor Icons (https://phosphoricons.com/)

## Implementation Notes

When implementing these designs:
- Use the exact font sizes, weights, and spacing specified
- Follow the color palette strictly for consistency
- Implement all hover and disabled states for interactive elements
- Use Phosphor Icons for all iconography
- Maintain proper visual hierarchy through typography scale
